# alone-admin
alone-admin是基于vue+element-ui开发的后台管理系统，至于其他的介绍，看下面的图自行脑补哈。

PS：alone-admin为新手作品，大佬勿喷。

![](http://qdfvid0sp.bkt.clouddn.com/alone-admin.png)
## 项目运行
```
git clone git@github.com:pmb2020/alone-admin.git //克隆项目到本地

cd alone-admin  //进入项目

npm install  //npm安装依赖

npm run server //运行项目
```
